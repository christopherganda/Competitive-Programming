                                                                                                  int myComp(const void* a, const void* b)
                                                                                                  {
                                                                                                  struct Edge* a1 = (struct Edge*)a;
                                                                                                  struct Edge* b1 = (struct Edge*)b;
                                                                                                  return a1->weight > b1->weight; <----THIS
                                                                                                  }

                                                                                                  This returns 1 if a1.weight is greater than b1.weight and 0 otherwise.

                                                                                                  What one must return is:

                                                                                                  0: if both values are equal

                                                                                                  <0: if a1.weight is less than b1.weight

                                                                                                  >0: if a1.weight is more than b1.weight
                                                                                                  acc to formal def. of qsort

                                                                                                      AllergicToBitches surbhijain93 • 4 months ago

                                                                                                      Very good observation.
                                                                                                      return a1.weight > b1.weight; will always return 1 or 0 based on a1 weight is more than b1's or not. It will never return value<0.
                                                                                                      Case 1: a1->weight > b1->weight -> myComp will return 1 : qsort will place a1 after b1 ie. b1, a1
                                                                                                      Case 2: a1->weight < b1->weight -> myComp will return 0 : order will remain unchanged ie. a1, b1
                                                                                                      Case 3: a1->weight == b1->weight -> myComp will return 0 : order will remain unchanged ie. a1, b1

                                                                                                      But this kind of expression should be avoided. Better use return a1.weight - b1.weight;

                                                                                                      GeeksforGeeks Please change the expression. Its really confusing.
